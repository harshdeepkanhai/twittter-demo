## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- `ruby 2.7 (rbenv preferred)`
- `bundle` gem installed
- `rails` gem installed
- `yarn` installed

### Setup
- clone the repo

### Install
- `cd` into the repo
- run `bundle install`
- run `yarn`
- run `rails db:migrate`
- run `rails server`
### Usage
- open `localhost:3000` on your browser to see the blog working
